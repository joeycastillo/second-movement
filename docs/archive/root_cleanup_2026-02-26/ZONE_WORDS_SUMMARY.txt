🎮 DISPLAY INTERACTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  On face activation:
    1. Check if new day → update streak (increment or reset)
    2. Get current metric value → determine word level (0-3)
    3. Select 2 random words from level's 12-word array
    4. Apply mixing (20% cross-zone, 10% wildcard)
    5. Display first word (Mode 0)

  ALARM button press:
    1st press → Show second word (Mode 1)
    2nd press → Show stats: "E Lv2 3d" (Mode 2)
    3rd press → Cycle back to first word (Mode 0)

  Stats format:
    [Zone Letter] [Space] Lv[Level] [Space] [Streak]d
    Example: "E Lv2 3d" = Emergence, Level 2, 3-day streak
